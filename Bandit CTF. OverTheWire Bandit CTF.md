## 📋 Executive Overview
This repository contains a comprehensive technical breakdown of the **OverTheWire (OTW) Bandit** wargame. The focus of this project is to demonstrate mastery of the Linux command line, secure communication protocols, and administrative forensics.

### 🛠️ Core Competencies Demonstrated
* **Data Extraction:** Locating hidden data in files (spaces in names, hidden files, non-human readable formats).
* **Cryptography & Encoding:** Decoding Base64, ROT13, and managing SSL/TLS certificates via `openssl`.
* **Network Analysis:** Using `nmap` and `netcat` to identify open ports and communicate with socket-based services.
* **Privilege Escalation:** Exploiting setuid permissions and analyzing `cron` jobs to understand system-level execution.
